# Practice API

## Overview
A sample API project for learning and development purposes.

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
```bash
npm install
```

### Running the Server
```bash
npm start
```

## API Endpoints
- `GET /api/health` - Health check
- `POST /api/data` - Create data
- `GET /api/data/:id` - Retrieve data
- `PUT /api/data/:id` - Update data
- `DELETE /api/data/:id` - Delete data
